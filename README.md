# Ethersoft

Luxury multi-page marketing site for Ethersoft — intelligent software for the cloud and the physical world.

Static HTML/CSS/JS (no build step). Hosted on GitHub Pages.

## Pages
- **index.html** — Home hub linking to every division.
- **cloud-storage.html** — Cloud Storage Management (AI Storage Systems, Storage Optimizator, GCNV Storage Manager + FAQ).
- **smart-cart.html** — Ethersoft Smart Cart AI (shopper experience + retailer/CPG advantage + concept gallery).
- **company.html** — Company / profile.
- _College Research_ — planned as a separate site in the future.

## To finish
Drop these files into `images/` (names must match):
- `profile.jpg` — your headshot (Company page).
- `cart-v1.png`, `cart-v2.png` — Smart Cart concept renders.

Then fill in `[Your Name]`, `[Your Title]`, and the bio in `company.html`.

## Local preview
Open `index.html` in a browser, or run `python3 -m http.server` in this folder.
